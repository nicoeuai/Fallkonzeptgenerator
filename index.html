Psychotherapie‑Bericht Generator

Erstellen Sie strukturiert Berichte nach PTV 3

Herzlich willkommen!

Mit diesem Tool können Sie einen strukturierten Bericht für Erst‑, Umwandlungs‑ oder Fortführungsanträge erstellen. Die Eingabemaske führt Sie Schritt für Schritt durch alle notwendigen Angaben entsprechend der gesetzlichen Vorgaben.

Antragstyp wählen

Bitte wählen Sie aus, welchen Antrag Sie stellen möchten.

1. Soziodemographische Daten

2. Symptomatik und subjektiver Leidensdruck

3. Psychischer Befund

Bitte beschreiben Sie für jeden Bereich die beobachteten Merkmale. Sie können Stichpunkte verwenden.

4. Somatischer Befund / Konsiliarbericht

5. Lebensgeschichte & funktionales Bedingungsmodell

6. Diagnosen

7. Behandlungsplan und Prognose

8. Zusätzliche Angaben bei Umwandlungs‑/Fortführungsanträgen

Dieser Abschnitt wird nur benötigt, wenn Sie einen Umwandlungs- oder Fortführungsantrag stellen. Bei Erstanträgen können Sie „Weiter“ klicken.

9. Zusammenfassung

Im folgenden Feld sehen Sie Ihren generierten Bericht. Sie können diesen kopieren oder als Textdatei herunterladen.
